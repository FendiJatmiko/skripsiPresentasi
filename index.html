rancang bangun media server cloud computing menggunakan plex pada docker

yang melatar belakangi pembuatan karya ilmiah ini adalah

meningkatnya pengguna komputer dan internet membuat kebutuhan pengolahan data media juga meningkat

terutama data multimedia seperti rekaman video dan audio yang berukuran relatif cukup besar

sedangkan kebanyakan device mobile memiliki ukuran penyimpanan yang relatif kecil

Rumusan Masalah

berdasarkan latar belakang masalah diatas maka rumusan masalah dalam penelitian ini adalah:
  1. Bagaimana menghasilkan layanan media server berbasis cloud computing
  2. bagaimana menghasilkan server enyimpanan yang dapat ditambah kapasitas penyimpanannya tanpa harus ada downtime.

RUang Lingkup

agar penelitian ini tidak meluas maka yang menjadi acuan dalam pengerjaan penelitian ini adalah :

menggunakan aplikasi plex sebagai manajemen media pada server dan media player pada sisi klien

jenis media yang dapat diputar berupa :
  • audio = AAC, ALAC, FLAC, MP3, M4A, WAV
  • video = MKV, MP4, MOV, FLV, 3gp
  • media server ini berbasis web
  • media server ini bersifat scalable

Tujuan

adapun tujuan dibuatnya penelitian ini sebagai berikut:
  1. Untuk mengimplementasikan plex media server dalam docker container
  2. membangun media server berasis cloud computing yang dapat diakses melalui jaringan nirkable
